It includes a class (RdlDowngradeEngine) that does all the heavy lifting of downgrading a report.

There is a sample Windows Forms application that you can use (or extend) to downgrade reports and display warnings / errors.

There is also a Report Server projet that contains reports with R2 elements in it so that you can experiment with the downgrade.

Most of the documentation is in french (see blog posts below). You can use Google Translate or better yet, learn french! ;-)
